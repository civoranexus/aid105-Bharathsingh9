## User Profile Data
- Age
- Annual family income
- State and district (from LGD)
- Category (SC/ST/OBC/General)
- Gender

## Scheme Data
- Scheme name
- Description and benefits
- Eligibility criteria
- Age limits
- Income limits
- Applicable state or level
- Category and gender eligibility

## Live Data Sources
- Local Government Directory (LGD) for administrative locations

## Data Integration Approach
The system follows a hybrid data integration approach by using live government
data where available and manually structured eligibility rules where live APIs
are not available.
